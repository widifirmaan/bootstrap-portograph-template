<h1 align="center">📽️ Portograph - Cinematic Wedding Portfolio</h1>

## 🚀 Features Overview

### ✨ Visual Experience
*   **Hero Showcase**: Immersive header with sleek animations.
*   **Fluid Animations**: Smooth transitions and hover effects for a premium feel.
*   **Interactive Gallery**: Dedicated section to highlight your best visual work.

### 🛠️ Practical Functionality
*   **Package Management**: Clear and attractive layout for service tiers.
*   **Instant WhatsApp Ordering**: Direct-to-chat booking system for higher conversions.
*   **Social Integration**: Seamlessly link your Instagram and YouTube portfolios.

### ⚡ Technical Excellence
*   **Super Lightweight**: Optimized for lightning-fast loading speeds.
*   **Fully Responsive**: Flawless experience across desktop, tablet, and mobile.
*   **SEO Optimized**: Built with semantic HTML for maximum search visibility.

---

## 🛠️ Tech Stack

### Core Technologies
*   **HTML5**: Modern semantic structure.
*   **CSS3**: Custom styling with advanced visual effects.
*   **Vanilla JavaScript**: High-performance interactive elements.

### Architecture
*   **No Heavy Frameworks**: Pure code for zero overhead and maximum speed.
*   **Vendor Scripts**: AOS (Animate on Scroll), GLightbox, Swiper, and Typed.js.

## 📦 Getting Started

### Prerequisites
*   A modern web browser.
*   Basic knowledge of HTML/CSS for customization.

### Installation
1.  **Clone the repository**
    ```bash
    git clone https://github.com/widifirmaan/Portograph-Template.git
    ```
2.  **Run Locally**
    *   Simply open `index.html` in your browser.
    *   Or use a local server like Live Server (VS Code) or `python3 -m http.server`.

3.  **Deploy**
    *   Upload to GitHub Pages, Vercel, Netlify, or any static hosting provider.
